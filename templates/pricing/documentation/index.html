A) HTML Structure - top

The CSS Responsive Pricing Tables Mega Pack consists of 30 different unique pricing table themes.

This pack is based on bootstrap framework. Follow the link to read more about twitter bootstrap framework.

The html structure is presented beyond:

	< div class="span3">
          < div class="for_header">
            < h4 class="gr_grey">Basic plan< /h4>
            < div class="price">
              < p>$2.99 < span>/month< /span> < /p>
            < / div>
          < / div>
          < div class="for_border">
            < p>< span>200Mb Monthly traffic < /p>
            < p>< span>10Gb disk space< /p>
            < p>< span>5 Domains< /p>
            < p>< span>5 Subdomains< /p>
            < p>Unlimited email support< /p>
            < p>PHP 5 Support< /p>
            < p>MySql support< /p>
            < p>< a class="btn" href="#">Sign Up< /p>
          < /div>
        < /div>

Where the span3 is a holder for a single pricing table which is a bootstrap's responsive grid class; for_header is a holder for the pricing table's header; for_border is a holder for a "body" of the table.

If you want to add more items in the pricing tables list, just add anoter < p > < / p > tag in the for_border section as in the example above.

B) CSS Files and Structure - top

The package contains 30 diferent .css files for each individual table style. The files are easy to edit. Use the structure listed above to navigate the element you want to edit. For example if you want to edit the header color of the table in corresponding css file find the .for_header rule and edit the "background-color" property. This works for all other elements.

C) Usage & Installation - top

To copy tables to your project you have to make sure that you're copying the bootstrap's css files as well.

Let's make it step by step:

1) Copy the following lines to your projects < head > section: 

< link href="css/bootstrap.css" rel="stylesheet">

< link href="css/bootstrap-responsive.css" rel="stylesheet">

< link data-style="myStylez" href="css/style_1.css" rel="stylesheet">
css/style_1.css is the stylesheet associated with theme's number. By changing the number "_1" you're actually pointing different template.

2) Copy the pricing table structure that is placed inside "container" div and goes right after the "spacer" div.

And that's it.
